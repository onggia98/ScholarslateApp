CORE FEATURES
08

Sáu tính năng làm nên ScholarSlate.

UC10 · UC11

Auto-fetch từ arXiv

Scheduler 06:00 quét topic active, gọi arXiv Atom API, delay 1500ms / request, UPSERT theo arxiv_id.

UC11

AI Summary + Quality Score

Groq llama-3.1-8b tóm tắt abstract, sinh quality_score ∈ [0–10]. Hiển thị ngay trong paper card.

UC14

Semantic Recommendation

HNSW cosine search trên vector(384), top-10 paper liên quan, Caffeine cache TTL 1h.

UC05 · UC12

Full-text Search + Match

GIN index + plainto_tsquery cho user search · phraseto_tsquery cho topic matching.

UC07 · UC08

Favorite & Notification

Lưu paper yêu thích idempotent, badge số chưa đọc, NEW_PAPER khi có paper mới khớp topic.

UC15

Trend Analytics

GROUP BY YYYY-MM trên PAPER × PAPER_TOPIC. 24 tháng gần nhất, biểu đồ đường theo topic.

QUYẾT ĐỊNH KỸ THUẬT · KEY DECISIONS
10

5 quyết định kỹ thuật quan trọng.

01
VectorUserType thay AttributeConverter
✗ setString() → PostgreSQL type mismatch
✓ PGobject.type="vector" → đúng OID, không lỗi
02
Manual Mapper thay MapStruct
✗ Spring Boot 4.x component scan conflict
✓ @Component manual mapper — kiểm soát hoàn toàn
03
arXiv delay 350ms → 1500ms
✗ 350ms → HTTP 429 trên Railway logs
✓ 1500ms — không còn rate limit error
04
BAAI/bge thay all-MiniLM-L6-v2
✗ HF phân loại sai pipeline (sentence-similarity)
✓ feature-extraction, cùng 384 dims, không đổi schema
05
REQUIRES_NEW per paper — Transaction isolation
✗ Outer @Transactional → lỗi 1 paper rollback toàn batch  ·  HikariCP pool exhaustion (max 20 conn)   ✓ REQUIRES_NEW: mỗi paper 1 transaction độc lập, outer không có @Transactional
