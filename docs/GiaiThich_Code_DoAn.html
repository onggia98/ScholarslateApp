3. SecurityConfig — Cấu hình bảo mật toàn hệ thống

security/SecurityConfig.java

Class này là trung tâm của toàn bộ security layer — định nghĩa ai được phép làm gì, token được xác thực thế nào, và CORS hoạt động ra sao.

SecurityConfig.java 
@Configuration
@EnableWebSecurity
@EnableMethodSecurity          // cho phép @PreAuthorize trên method
@RequiredArgsConstructor        // Lombok inject qua constructor
public class SecurityConfig {

  @Bean
  public SecurityFilterChain securityFilterChain(HttpSecurity http) {
    http
      .csrf(AbstractHttpConfigurer::disable) // tắt CSRF — stateless JWT không cần
      .cors(cors -> cors.configurationSource(corsConfigurationSource()))
      .sessionManagement(s -> s.sessionCreationPolicy(STATELESS)) // không tạo session
      .authorizeHttpRequests(auth -> auth
        .requestMatchers(HttpMethod.OPTIONS, "/**").permitAll() // CORS preflight
        .requestMatchers("/api/actuator/health").permitAll()     // Railway health check — không có JWT khi Railway probe
        .requestMatchers("/api/auth/**").permitAll()              // login/register
        .requestMatchers("/api/admin/**").hasRole("ADMIN")        // admin gate
        .anyRequest().authenticated()                              // còn lại cần token
      )
      // 401/403 custom JSON — tránh redirect HTML từ Spring Security mặc định
      .exceptionHandling(ex -> ex
        .authenticationEntryPoint((req, res, e) -> {
          res.setStatus(401);
          res.setContentType("application/json");
          res.getWriter().write("{\"success\":false,\"message\":\"Unauthorized\",\"data\":null}");
        })
        .accessDeniedHandler((req, res, e) -> {
          res.setStatus(403);
          res.setContentType("application/json");
          res.getWriter().write("{\"success\":false,\"message\":\"Forbidden\",\"data\":null}");
        })
      )
      .addFilterBefore(jwtAuthenticationFilter, UsernamePasswordAuthenticationFilter.class);
    return http.build();
  }
Tại sao CSRF bị disable? CSRF protection bảo vệ session-based auth. Hệ thống này dùng JWT stateless — không có session, không có cookie chứa CSRF token. Disable là đúng, không phải lỗ hổng.

JwtUtil — tạo và xác thực token

JwtUtil.java 
@Component
public class JwtUtil {
  private final SecretKey secretKey;
  private final long expirationMs;

  public JwtUtil(
      @Value("${jwt.secret}") String secret,
      @Value("${jwt.expiration-ms}") long expirationMs) {
    // Keys.hmacShaKeyFor: require secret bytes >= 256 bit (32 bytes)
    this.secretKey = Keys.hmacShaKeyFor(secret.getBytes(StandardCharsets.UTF_8));
    this.expirationMs = expirationMs;
  }

  // Tạo token chứa userId (subject), email, role, issuedAt, expiration
  public String generateToken(UUID userId, String email, String role) {
    Date now = new Date();
    return Jwts.builder()
      .subject(userId.toString())
      .claim("email", email)   // frontend hiển thị không cần gọi API profile
      .claim("role", role)
      .issuedAt(now)
      .expiration(new Date(now.getTime() + expirationMs))
      .signWith(secretKey)     // HMAC-SHA256 tự động detect từ key type
      .compact();
  }

  public boolean isTokenValid(String token) {
    try {
      Claims claims = extractClaims(token);
      return claims.getExpiration().after(new Date());  // chưa hết hạn
    } catch (JwtException | IllegalArgumentException e) {
      return false;   // token lỗi, giả mạo, hoặc hết hạn
    }
  }
}

JwtAuthenticationFilter — chạy mỗi request

JwtAuthenticationFilter.java 
public class JwtAuthenticationFilter extends OncePerRequestFilter {

  @Override
  protected void doFilterInternal(...) {
    String token = extractToken(request); // từ header "Authorization: Bearer ..."

    if (StringUtils.hasText(token) && jwtUtil.isTokenValid(token)) {
      UUID userId = jwtUtil.extractUserId(token);
      String role  = jwtUtil.extractRole(token);

      // principal = UUID userId — controllers lấy qua @AuthenticationPrincipal
      var auth = new UsernamePasswordAuthenticationToken(
        userId, null,
        List.of(new SimpleGrantedAuthority("ROLE_" + role))
      );
      SecurityContextHolder.getContext().setAuthentication(auth);
    }
    // Không throw exception — request không có token vẫn đi tiếp,
    // SecurityFilterChain sẽ từ chối nếu endpoint yêu cầu auth.
    filterChain.doFilter(request, response);
  }
}
Tại sao dùng UUID làm principal thay vì UserDetails? Với stateless JWT, không cần load UserDetails từ DB mỗi request. Filter đọc userId trực tiếp từ token → inject vào SecurityContext. Controllers nhận qua @AuthenticationPrincipal UUID userId. Tiết kiệm 1 DB round-trip mỗi request.

5. Entity Paper — trái tim của hệ thống

entity/Paper.java
Paper.java 
@Entity @Table(name = "paper")
@Getter @Setter @NoArgsConstructor @AllArgsConstructor @Builder
public class Paper {

  @Id @GeneratedValue(strategy = GenerationType.UUID)
  private UUID id;

  @Column(name = "arxiv_id", unique = true, length = 50)
  private String arxivId;  // e.g. "2301.00001"

  // "abstract" là keyword của Java → dùng abstractText ở Java, map sang cột "abstract" ở DB
  @Column(name = "abstract", columnDefinition = "TEXT")
  private String abstractText;

  /**
   * vector(384) — không thể dùng AttributeConverter vì JDBC setString() gây lỗi:
   * "column is of type vector but expression is of type character varying"
   * Fix: dùng VectorUserType (PGobject.type="vector") — PostgreSQL nhận đúng OID
   */
  @Type(VectorUserType.class)
  @Column(columnDefinition = "vector(384)")
  private float[] embedding;

  // Trạng thái pipeline: "PENDING" | "DONE" | "FAILED"
  @Column(name = "processing_status", nullable = false)
  @Builder.Default
  private String processingStatus = "PENDING";

  @Column(name = "retry_count")
  @Builder.Default
  private Integer retryCount = 0;

  // FK tự tham chiếu — paper gốc khi is_duplicate = true
  @ManyToOne(fetch = FetchType.LAZY)
  @JoinColumn(name = "original_paper_id")
  private Paper originalPaper;

  // LAZY — tránh N+1 khi list paper (topics chỉ cần khi xem chi tiết)
  @ManyToMany(fetch = FetchType.LAZY)
  @JoinTable(name = "paper_topic",
    joinColumns = @JoinColumn(name = "paper_id"),
    inverseJoinColumns = @JoinColumn(name = "topic_id"))
  @Builder.Default
  private List<Topic> topics = new ArrayList<>();
}

6. VectorUserType — Bridge giữa Java và pgvector

converter/VectorUserType.java

Đây là class kỹ thuật quan trọng nhất trong dự án — giải quyết vấn đề incompatibility giữa Hibernate 7.x và pgvector.

Tại sao không dùng AttributeConverter? AttributeConverter<float[], String> dùng JDBC setString() → PostgreSQL nhận character varying thay vì vector → lỗi type mismatch. Phải dùng UserType để set PGobject.type = "vector" — JDBC driver gửi đúng PostgreSQL OID.
VectorUserType.java 
public class VectorUserType implements UserType<float[]> {

  @Override
  public int getSqlType() { return Types.OTHER; }  // pgvector = custom → OTHER

  // ── Đọc từ DB → Java ────────────────────────────────────────────
  @Override
  public float[] nullSafeGet(ResultSet rs, int position, ...) {
    String value = rs.getString(position);     // DB trả "[0.1,0.2,...,0.384]"
    return value == null ? null : parseVector(value);
  }

  // ── Ghi vào DB → PostgreSQL ──────────────────────────────────────
  @Override
  public void nullSafeSet(PreparedStatement st, float[] value, int index, ...) {
    if (value == null) {
      st.setNull(index, Types.OTHER);
    } else {
      PGobject pgObj = new PGobject();
      pgObj.setType("vector");              // ← KEY: PostgreSQL nhận đúng OID
      pgObj.setValue(toVectorString(value)); // float[] → "[0.1,0.2,...,0.384]"
      st.setObject(index, pgObj);
    }
  }

  // ── Helpers ──────────────────────────────────────────────────────
  private String toVectorString(float[] arr) {
    StringBuilder sb = new StringBuilder("[");
    for (int i = 0; i < arr.length; i++) {
      if (i > 0) sb.append(",");
      sb.append(arr[i]);
    }
    return sb.append("]").toString();  // "[0.1,0.2,...,0.384]"
  }

  private float[] parseVector(String s) {
    // Trim brackets: "[0.1,0.2]" → "0.1,0.2" → split → float[]
    String trimmed = s.trim().replaceAll("^\\[|\\]$", "");
    return Arrays.stream(trimmed.split(","))
      .map(String::trim)
      .map(Float::parseFloat)
      ...
  }
}

7. PaperMapper — Manual mapper không MapStruct

mapper/PaperMapper.java
Tại sao không dùng MapStruct? MapStruct generate code lúc compile-time bằng annotation processor. Spring Boot 4.x thay đổi component scan order → MapStruct-generated class không được register đúng lúc → NoSuchBeanDefinitionException. Giải pháp: viết mapper thủ công, đơn giản hơn và kiểm soát hoàn toàn.
PaperMapper.java 
@Component   // Spring bean, inject bình thường qua @RequiredArgsConstructor
public class PaperMapper {

  // Overload 1: có user context — isFavorite được chỉ định rõ ràng
  public PaperResponse toResponse(Paper paper, boolean isFavorite) {
    List<String> topicNames = paper.getTopics() == null ? List.of()
      : paper.getTopics().stream()
          .map(Topic::getName)
          .distinct()    // phòng khi cùng topic name xuất hiện 2 lần
          .toList();

    return PaperResponse.builder()
      .id(paper.getId())
      .title(paper.getTitle())
      .abstractText(paper.getAbstractText()) // field Java ≠ tên JSON
      ...
      // embedding KHÔNG mapping → không bao giờ ra ngoài API
      .isFavorite(isFavorite)
      .build();
  }

  // Overload 2: không có user context (admin/recommendation)
  public PaperResponse toResponse(Paper paper) {
    return toResponse(paper, false);
  }

  // Batch: favoritedIds là Set để O(1) contains() — không N+1
  public List<PaperResponse> toResponseList(List<Paper> papers, Set<UUID> favoritedIds) {
    return papers.stream()
      .map(p -> toResponse(p, favoritedIds.contains(p.getId())))
      .toList();
  }
}

Tại sao embedding không được map?

Embedding là 384 floats (~1500 bytes khi serialize). Expose ra API sẽ: (1) tăng payload size vô ích, (2) leak thông tin nội bộ về model. PaperResponse không có field embedding → Mapper không thể vô tình set.

// <=> là pgvector cosine DISTANCE operator (0=identical, 1=orthogonal)
// Threshold 0.05 = similarity > 95% — rất giống nhau
// Window 90 ngày — không so với paper quá cũ
@Query(value = """
  SELECT * FROM paper
  WHERE processing_status = 'DONE' AND is_duplicate = false
  AND id != :excludeId
  AND published_at >= :windowStart
  AND embedding IS NOT NULL
  AND embedding <=> CAST(:embedding AS vector) < 0.05
  ORDER BY embedding <=> CAST(:embedding AS vector)
  LIMIT 1
  """, nativeQuery = true)

Topic matching (UC12)

// phraseto_tsquery (khác plainto): match exact phrase theo thứ tự
// "large language model" → phải xuất hiện nguyên cụm, không bị tách
// Index: idx_paper_fts_TOPIC (GIN) — chỉ title+abstract, KHÔNG có authors
// (phải khớp hoàn toàn với expression trong CREATE INDEX, nếu include authors → seq scan)
@Query(value = """
  SELECT COUNT(*) > 0 FROM paper
  WHERE id = :paperId
  AND to_tsvector('english', coalesce(title,'') || ' ' || coalesce(abstract,''))

Tại sao phraseto vs plainto? plainto_tsquery('english', 'large language model') match cả "large model with language skills" (AND logic, không theo thứ tự). phraseto_tsquery match "large language model" theo thứ tự chính xác. Keywords như "transformer", "attention mechanism", "large language model" cần exact phrase match để tránh false positive.

      return PagedResponse.of(

          .map(p -> paperMapper.toResponse(p, favIds.contains(p.getId()))));
    }
    ...
  }

  @Transactional(readOnly = true)
  public List<PaperResponse> getRecommendations(UUID paperId) {
    var paper = paperRepository.findById(paperId).orElseThrow(...);
    if (paper.getEmbedding() == null) return List.of();

    // float[] → "[0.1,...,0.384]" → native query CAST(:embedding AS vector)

public List<Paper> fetchByKeyword(String keyword) {

  // Exponential backoff: 0s → 2s → 4s (tối đa 2 lần retry)

      if (attempt > 0) Thread.sleep((long) Math.pow(2, attempt) * 1000L);
      xml = restClient.get().uri(url).retrieve().body(String.class);
      break;

      if (attempt == 2) return List.of(); // hết lần retry
    }

  Thread.sleep(requestDelayMs); // 1500ms delay — phòng 429
  return parseAtomXml(xml);
}

  // DOM parser duyệt <entry> elements trong Atom feed

  for each <entry> {
    // arxiv_id: "http://arxiv.org/abs/2301.00001v1"
    // → extract "2301.00001" (bỏ domain và version suffix)
    arxivId = raw.replaceAll(".*/abs/", "").replaceAll("v\\d+$", "");

    // authors: gom từ <author><name>...</name></author> thành "A, B, C"
    authors = authors == null ? name : authors + ", " + name;
  }

  return papers; // Paper với processingStatus="PENDING", retryCount=0
}

      return raw.stream().map(this::extractEmbedding).toList();

    } catch (HttpServerErrorException e) {
      if (attempt == 0 && e.getStatusCode().value() == 503) {
        // Model cold start (HF ngủ sau ~15 phút không dùng) → chờ 30s
        Thread.sleep(retryWaitMs);
      }
    }
  }
}

private float[] meanPool(List<List<Number>> tokenEmbeddings) {
  // shape [seq_len, hidden_size] → average over seq_len dimension → [hidden_size]
  float[] result = new float[dims];
  for (var token : tokenEmbeddings)
    for (int d = 0; d < dims; d++) result[d] += token.get(d).floatValue();
  for (int d = 0; d < dims; d++) result[d] /= tokenEmbeddings.size();
  return result;
}

public record GroqResult(String summary, Float qualityScore, String error) {
  public boolean isValid() { return error == null; }

public GroqResult summarize(String title, String abstractText) {
  for (int attempt = 0; attempt <= 1; attempt++) {
    try {
      var body = Map.of(
        "model", model,
        "response_format", Map.of("type", "json_object"),  // buộc trả JSON
        "messages", List.of(Map.of("role", "user", "content", prompt))
      );
      var response = restClient.post().uri(GROQ_API).body(body).retrieve().body(Map.class);
      Thread.sleep(callDelayMs); // 2s delay giữa các calls
      return parseResponse(response);

    } catch (Exception e) {
      if (e.getMessage().contains("429")) {
        // Rate limit: KHÔNG retry ngay — Retry Scheduler xử lý sau 30 phút
        return new GroqResult(null, null, "Groq rate limit");
      }
      // 5xx/timeout: retry 1 lần sau 5s
      if (attempt == 1) return new GroqResult(null, null, e.getMessage());
      Thread.sleep(transientRetryDelayMs);
    }
  }
}

private GroqResult parseResponse(Map<?,?> response) {
  // Groq response: choices[0].message.content = "{\"summary\":\"...\",\"quality_score\":7.5}"
  var content = choices[0].message.content;  // String JSON
  var json = OBJECT_MAPPER.readValue(content, Map.class);

  String summary = (String) json.get("summary");
  float score = ((Number) json.get("quality_score")).floatValue();

  // Validate: summary không rỗng, ≤ 2000 chars; score ∈ [0.0, 10.0]
  if (summary == null || summary.isBlank())
    return new GroqResult(null, null, "Empty summary");
  if (score < 0 || score > 10)
    return new GroqResult(null, null, "score out of range: " + score);

  return new GroqResult(summary, score, null);
}

PaperPipelineService.java — Orchestrator 
@Transactional(propagation = Propagation.REQUIRES_NEW)
public boolean process(Paper paper) {
  // Idempotency check — arXiv có thể trả cùng paper cho nhiều keywords
  if (paperRepository.existsByArxivId(paper.getArxivId())) return false;

  Paper saved = paperRepository.save(paper); // cần ID cho duplicate query
  runPipeline(saved);
  return true;
}

private void runPipeline(Paper paper) {

  // ── Step 1: Embedding ─────────────────────────────────────────────
  float[] embedding = embeddingService.embed(title + " " + abstract);
  if (embedding == null) { markFailed(paper, "Embedding failed"); return; }
  paper.setEmbedding(embedding);

  // ── Step 2: Duplicate detection (cosine distance < 0.05) ─────────
  String embStr = VectorUtils.toVectorString(embedding);
  var dup = paperRepository.findDuplicate(paper.getId(), embStr, now().minusDays(90));
  if (dup.isPresent()) {
    paper.setIsDuplicate(true);
    paper.setOriginalPaper(dup.get());
    paper.setProcessingStatus("DONE");
    paperRepository.save(paper);
    return; // duplicate không cần Groq hay topic matching
  }

  // ── Step 3: Groq summary + quality_score ─────────────────────────
  var result = groqService.summarize(paper.getTitle(), paper.getAbstractText());
  if (!result.isValid()) { markFailed(paper, result.error()); return; }
  paper.setSummary(result.summary());
  paper.setQualityScore(result.qualityScore());

  // ── Step 4: Topic matching + Notification ────────────────────────
  matchTopicsAndNotify(paper, topicRepository.findByIsActiveTrue());

  // ── Step 5: DONE ─────────────────────────────────────────────────
  paper.setProcessingStatus("DONE");
  paperRepository.save(paper);
}

private void matchTopicsAndNotify(Paper paper, List<Topic> topics) {
  for (Topic topic : topics) {
    boolean matched = Arrays.stream(topic.getKeywords().split(","))
      .map(String::trim)
      .anyMatch(kw -> paperRepository.matchesTopic(paper.getId(), kw));

    if (matched) {
      paperTopicRepository.insertIfNotExists(paper.getId(), topic.getId()); // ON CONFLICT DO NOTHING
      notificationRepository.insertIfNotExists(topic.getUser().getId(), paper.getId(), msg);
    }
  }
}
REQUIRES_NEW — Transaction isolation quan trọng Mỗi paper chạy trong transaction riêng. Nếu Groq trả lỗi cho paper thứ 5, transaction đó rollback, nhưng paper 1–4 đã COMMIT độc lập. Outer scheduler không có @Transactional để tránh HikariCP pool exhaustion khi batch 100+ papers.

// Dev: VITE_API_BASE_URL unset → dùng '/api' → Vite proxy → localhost:8081 (no CORS)

const API_BASE = import.meta.env.VITE_API_BASE_URL ?? '/api';

async function request<T>(path: string, options: RequestInit = {}): Promise<T> {
  const token = getStoredToken();        // từ localStorage/sessionStorage
  const res = await fetch(API_BASE + path, {
    ...options,
    headers: {
      'Content-Type': 'application/json',
      ...(token ? { Authorization: 'Bearer ' + token } : {}),
    },
  });

  if (res.status === 401) throw new ApiError(401, 'Unauthorized');
  if (res.status === 204) return undefined as T; // DELETE returns 204 No Content

  const body = await res.json();
  if (!res.ok) throw new ApiError(res.status, body?.message);

  // Unwrap ApiResponse<T> → .data
  // Backend luôn bọc trong { success, message, data: T }
  if ('data' in body) return body.data as T;
  return body as T;   // fallback nếu backend trả raw JSON
}

// Decode JWT payload mà không cần thư viện ngoài
export function decodeJwt(token: string): JwtPayload | null {
  const parts = token.split('.');     // header.payload.signature
  if (parts.length < 2) return null;

  let b64 = parts[1]                  // lấy payload (base64url)
    .replace(/-/g, '+')              // base64url → base64 standard
    .replace(/_/g, '/');
  while (b64.length % 4) b64 += '=';  // padding

  // Decode: base64 → binary string → percent-encoded → decodeURIComponent → JSON
  const json = decodeURIComponent(
    atob(b64).split('')
      .map(c => '%' + ('00' + c.charCodeAt(0).toString(16)).slice(-2))
      .join('')
  );
  return JSON.parse(json) as JwtPayload;
}

// Từ token → User object (email, initials cho avatar, role)
export function getUserFromToken(token: string): User {
  const payload = decodeJwt(token);
  const email = payload?.email || payload?.sub || 'User';
  const role  = payload?.role === 'ADMIN' ? 'ADMIN' : 'USER';
  return { email, initials: emailToInitials(email), role };
}

// emailToInitials: "john.doe@example.com" → "JD"
export function emailToInitials(email: string): string {
  const local = email.split('@')[0];
  const parts = local.split(/[._\-+]+/).filter(Boolean);
  if (parts.length >= 2) return (parts[0][0] + parts[1][0]).toUpperCase();
  return parts[0]?.slice(0, 2).toUpperCase() || 'U';
}

@Scheduled(cron = "${scheduler.main.cron:0 0 6 * * *}")
public void runDailyFetch() {
  var activeTopics = topicRepository.findByIsActiveTrue();
  if (activeTopics.isEmpty()) { log.info("No active topics"); return; }

  // LinkedHashSet giữ insertion order và dedup:
  // 2 users cùng có topic "machine learning" → chỉ fetch 1 lần
  Set<String> uniqueKeywords = new LinkedHashSet<>();
  for (var topic : activeTopics) {
    Arrays.stream(topic.getKeywords().split(","))
      .map(String::trim).filter(k -> !k.isEmpty())
      .forEach(uniqueKeywords::add);
  }

  for (String keyword : uniqueKeywords) {
    var papers = arxivFetchService.fetchByKeyword(keyword);
    for (var paper : papers) {
      try {
        paperPipelineService.process(paper); // REQUIRES_NEW — lỗi 1 paper không stop batch
      } catch (Exception e) {
        log.error("Pipeline error for arxivId={}: {}", paper.getArxivId(), e.getMessage());
      }
    }
  }
}
